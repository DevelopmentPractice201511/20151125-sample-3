# よく使われるコマンド
## ファイル操作

- :w    上書き保存
- :q    編集終了
- :q!   強制的に終了（編集されていても保存確認せず終了）
- :wq   保存して終了
- ZZ    同上

- :e [file-name]   ファイルをコマンドで指定して編集
- :E               vim上でファイルリストを開いてファイルを指定し、編集作業

- :bd   カレントのバッファーを閉じる
- :bn   次のバッファーに移る
- :bp   前のバッファーに移る
 
## 標準モード時
- j   上へ（カーソル移動）
- k   下へ（カーソル移動）
- h   左へ（カーソル移動）
- l   右へ（カーソル移動）
